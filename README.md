Описание программы
Программа демонстрирует работу корутин в стандарте C++20 и состоит из двух независимых частей, каждая из которых показывает различные аспекты использования корутин.

Часть 1: Демонстрация простой корутины
Цель:
Показать базовую структуру корутины на примере вывода заданного числа с дополнительными вычислениями.

Реализация:
Создана структура SimplePromise для управления состоянием корутины

Реализована корутина print_number(), которая принимает число в качестве параметра

Внутри корутины выполняется вывод:

Исходного числа: 455

Квадрата числа: 455² = 207025

Куба числа: 455³ = 94196375

Ключевые элементы:
SimplePromise::get_return_object() - возвращает handle корутины

std::suspend_always initial_suspend() - корутина приостанавливается при старте

std::suspend_always final_suspend() - корутина приостанавливается при завершении

co_return - завершение работы корутины

Часть 2: Корутина с анимированным прогрессбаром
Цель:
Показать практическое применение корутин для создания анимированного прогрессбара с персонализированным заполнением.

Реализация:
Создана структура Promise для управления прогрессом выполнения

Реализована корутина simulate_work(), имитирующая длительную операцию

Разработана функция draw_progress_bar() для отрисовки прогрессбара

Особенности прогрессбара:
Ширина: 50 символов

Имя для заполнения: "НАСТЯ"

Заполнение: циклическое повторение символов имени "Н", "А", "С", "Т", "Я"

Информация: отображается процент выполнения и текущий шаг (например: 45.0% (45/100))

Анимация: обновление на месте с помощью символа \r
